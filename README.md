A simple and modern **To-Do List Frontend** built with JavaScript.  
This project works together with the backend repository:  
🔗 **[todo-s-backend](https://github.com/Andreastephgm/todo-s-backend)**

---

## 📌 Features
- Create, edit, and delete tasks.
- Mark tasks as completed.
- Persistent data storage using the backend API.
- Responsive design for mobile and desktop.

---

## 🚀 Tech Stack
- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **Backend:** [todo-s-backend](https://github.com/Andreastephgm/todo-s-backend) (Node.js + Express)
- **Database:** MongoDB (configured in backend)

## ⚙️ Installation & Setup

1. **Clone the frontend repo**
```bash
git clone https://github.com/YOUR_USERNAME/todo-s-frontend.git
cd todo-s-frontend
```
2. Clone and run the backend
```bash
git clone https://github.com/Andreastephgm/todo-s-backend.git
cd todo-s-backend
npm install
npm start
```

3. Open the frontend

Simply open index.html in your browser.
Make sure the backend is running.
